## Background

I bought a StackOverflow Keyboard a few years ago, but had trouble setting it up

Fast forward to today, the Stackoverflow site is now in all practical terms obsolete and I still have an unconfigured keyboard.

I amd repurposing this Keyboard as the ASCKey.
More details in the README.md file  

## References
Instructions on how to setup the keyboard are in the /reference folder.
TheKey.json template has also been stored there.

## Functionality

The keys will be arranged as A, StackOverflow Icon Key, C Key in respective order.

A Key
- Pressing the key will result in opening a browser tab that navigates to https://claude.ai

Stackoverflow Key
- Pressing the key will result in opening a browser tab that navigates to https://stackoverflow.com

C Key
- Pressing the key will result in opening a browser tab that navigates to https://chatgpt.com

## Task
You are to implement the above functionality using the instructions in the /references folder
